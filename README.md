# BottleFi

🚧 **Work in Progress** 🚧

BottleFi is a decentralized application (DApp) that leverages blockchain technology to tackle the global issue of counterfeit wines. By enabling the minting, transferring, and invalidation of tokens, BottleFi ensures the authenticity of wine products, providing more confidence to consumers and retailers alike.

## Real World Use Case

Counterfeit products are a significant problem in many industries, with the wine industry being particularly affected. It is estimated that more than 20% of all wine sold worldwide is counterfeit, leading to massive revenue losses and damage to brand reputations. BottleFi addresses this issue by providing a secure, transparent, and immutable record of wine authenticity and ownership.

## Technology Stack

- **React**: For building the user interface
- **TypeScript (TS)**: For adding strong typing to JavaScript, enhancing code quality and maintainability
- **Ethers.js**: To interact with the Ethereum blockchain
- **IPFS**: For decentralized storage, ensuring that token meta data is stored securely and permanently
- **Tailwind CSS**: For styling the application with a utility-first CSS framework
- **Vite**: As a frontend build tool, enabling fast and efficient development experiences

## Features

- **Token Minting**: Create digital representations (tokens) of wine bottles on the blockchain.
- **Token Transfer**: Securely transfer ownership of wine tokens between parties.
- **Token Invalidation**: Invalidate tokens when bottles are consumed or decommissioned, maintaining an accurate ledger of active products.
